# 📉 Customer Churn Analysis (Python)

## 📌 Overview

This project analyzes customer churn behavior and builds a machine learning model to predict whether a customer will leave a service.

## 🎯 Objective

* Identify factors influencing customer churn
* Build a predictive model
* Provide actionable business insights

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib & Seaborn

## 🔄 Workflow

1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building (Logistic Regression / Random Forest)
5. Model Evaluation

## 📊 Key Insights

* Customers with high monthly charges are more likely to churn
* Long-term customers have lower churn rates
* Contract type significantly impacts retention
